# EcoTrace AI

EcoTrace AI is an AI-powered sustainability platform that helps users understand, track, and reduce their carbon footprint through intelligent insights and eco-friendly habit tracking.

Built for climate awareness, the platform combines carbon footprint monitoring, AI-powered sustainability coaching, receipt analysis, gamification, and community engagement to make sustainable living more accessible and rewarding.

---

## Features

### Carbon Footprint Tracking
- Track daily activities and their environmental impact
- Monitor sustainability progress over time
- Build eco-friendly habits through consistent tracking

### AI Sustainability Coach
- Personalized sustainability recommendations
- AI-generated environmental insights
- Actionable suggestions for reducing emissions

### Receipt & Image Analysis
- Analyze receipts and food images
- Identify high-impact consumption patterns
- Receive greener alternatives and recommendations

### Gamification
- Earn XP through sustainable actions
- Unlock achievements and milestones
- Maintain streaks and track progress

### Dashboard Analytics
- Sustainability score tracking
- Activity history and progress visualization
- Environmental impact overview

### Community
- Share achievements and milestones
- Engage with sustainability-focused users
- Encourage collective climate action

### User Profiles
- Personalized sustainability journey
- Progress and achievement tracking
- Customizable user experience

## Getting Started

### Prerequisites

- Node.js
- Gemini API Key

### Installation

```bash
git clone https://github.com/your-username/ecotrace-ai.git
cd ecotrace-ai
npm install
```

### Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run Locally

```bash
npm run dev
```
